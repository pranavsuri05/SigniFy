# SigniFy
SigniFy is an AI-powered real-time sign language translator that helps mute individuals communicate effortlessly in virtual meetings. The key feature is its seamless integration as a browser extension, starting with Google Meet and expanding to other platforms in the future. By leveraging real-time gesture recognition, SigniFy translates sign language into text displayed directly on the screen, making conversations more inclusive and accessible. This innovation bridges the communication gap for mute users, enhancing digital interactions without requiring additional hardware or apps.
